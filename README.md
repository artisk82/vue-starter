## Usage

### Step 1: Install the dependencies

```
npm install
npm install connect
npm install server-static
```

### Step 2: Run Gulp

```
gulp
```

### Step 3: Serve it

```
cd public
node server.js
```
and then hit http://localhost:8888

### Step 4: Login

```
username: admin@example.com
password: admin
```
